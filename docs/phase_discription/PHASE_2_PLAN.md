# План Фазы 2: Глубокое исследование и сравнительный анализ

> **Статус:** Активна
> **Цель:** Проведение масштабных экспериментов на реальных данных, сравнение архитектур (MLP/CNN vs KAN) и анализ интерпретируемости.

## 🎯 Основные цели фазы

1. **Запуск на реальных данных:** Переход от "smoke tests" к полноценному обучению моделей на всем массиве данных (50k+ осциллограмм).
2. **Data-Centric подход:** Глубокий анализ входных данных. Формирование гипотез о том, какие признаки (сырой сигнал, спектр, симметричные составляющие, комплексные числа) лучше всего подходят для KAN.
3. **Сравнительный анализ:** Честное сравнение KAN с классическими моделями (MLP, CNN, ResNet) по метрикам качества, количеству параметров и скорости инференса.
4. **Интерпретируемость:** Анализ того, *что* именно выучила KAN (визуализация функций активации).

- **Медленное обучение KAN:** KAN может учиться значительно медленнее MLP. Решение: использовать GPU для обучения, но оценивать инференс на CPU.
- **Переобучение:** KAN с большим `grid_size` склонен к переобучению. Решение: жесткая регуляризация и валидация.
